# Emir Can Onel

**Product-oriented full stack engineer.** I build SaaS platforms, mobile apps, and AI-powered tools — from first commit to production.

Currently working on **Algonel** (multi-tenant stock & sales management SaaS) and **Covera** (insurance operations platform).

---

### What I'm building

**[Algonel Admin Panel](https://emircanonel.vercel.app/en/work/ima-admin)** — Multi-tenant back-office system covering tenants, branches, warehouses, products, variants and audit workflows. Next.js + Supabase, multi-tenant data isolation.

**[Algonel Mobile App](https://emircanonel.vercel.app/en/work/ima-app)** — Branch operations app for Algonel. Orders, stock transfers, customer checkout, realtime activity. Expo + Supabase.

**[Covera](https://emircanonel.vercel.app/en/work/covera)** — Insurance operations platform. Customers, policies, collections, dashboards and RBAC with row-level security. Next.js + Supabase.

**[Mercan.ai](https://emircanonel.vercel.app/en/work/mercan-ai)** — AI video clipping platform. Next.js dashboard + FastAPI backend for job queues, render options, subtitles and short-form clip delivery.
