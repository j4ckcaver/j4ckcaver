# Emir Canonel

**Product-oriented full stack engineer.** I build SaaS platforms, mobile apps, and AI-powered tools — from first commit to production.

Currently working on **Algonel** (multi-tenant stock & sales management SaaS) and **Covera** (insurance CRM platform).

---

### What I'm building

**[Algonel](https://github.com/j4ckcaver)** — Multi-tenant SaaS for stock and sales management. Next.js + Supabase + React Native. Real business logic, multi-org architecture, mobile-first.

**[Covera](https://github.com/j4ckcaver)** — Insurance agency CRM. Policy tracking, agency workflows, built for real operators.

**[Daily Commit Tracker](https://github.com/j4ckcaver)** — Mobile app to track and protect GitHub streaks. Because streaks matter.

### Stack

```
Frontend   React · Next.js · React Native · TypeScript · Tailwind
Backend    Node.js · FastAPI · Supabase · PostgreSQL · MongoDB
```

### How I work

- Multi-tenant architecture from day one, not bolted on later
